# Vending Machine Simulator

## Pre-requisites
Prequisites to run this app are:
- Java 1.8
- Maven 3.x

## Run Tests
Run the following command in the project directory
```
mvn clean test
```

## Run App
Run the following command in the project directory
```
mvn exec:java -D"exec.mainClass"="com.lammyhill.vendingmachine.VendingMachineRunner"
```
